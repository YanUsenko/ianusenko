Fähigkeiten und Tools, die ich erlernt habe
Grafik
In der Letzen Zeit habe ich eine Vielzahl an Grafiksoftwares erprobt, zu meist benutze ich Adobe PS, Adobe Ai und natürlich Figma. Gerne würde es mich freuen und motivieren etwas neues zu erlernen.
Сoding
Mit HTML und CSS bin ich gut vertraut, JavaScript kommt dem sehr nahe. In meinen Arbeiten verwende ich den SASS-Präprozessor. Um meinen Code zu schreiben, verwende ich VS Code.
Ich sehe keine Schwierigkeit darin, eine Seite anpassungsfähig zu entwickeln, damit die diese auf jedem Display gut aussieht. Je nach Anforderung kann ich auch zum Layout der Blöcke Flexbox, Bootstrap oder Grid verwenden und einbauen.
Die Hauptaufgabe besteht darin, einen Code so zu schreiben, dass dieser in allen Browsern korrekt funktioniert.
Es würde mir eine Freude sein, meine Fähigkeiten weiter auszubauen, um neue Technologien und Programmiersprachen anzuwenden!
Über die Website
Seiteninhalt
Zuerst dachte ich darüber nach, womit ich diese Seite füllen kann. Nachdem ich die notwendigen Blöcke bestimmt habe, begann ich den Text zu schreiben.
Design
Die nächste Aufgabe bestand darin, ein Design zu erstellen, das leicht verständlich, minimalistisch genug, und doch nicht langweilig ist. Nachdem ich mehrere Varianten ausprobiert habe, entschied ich mich für ein weiches Farbschema und die Schriftart Roboto Mono, die für jedes Zeichen den gleichen Platz einnimmt und optisch einem Terminal oder alten Codierungsumgebungen ähnelt.
Code
Sehr gut, nun besteht die Aufgabe darin das erstellte Design für den Browser, in verständliche Sprachen zu übertragen. Wie immer begann alles mit dem Schreiben des HTML-Codes sowie dem Stylen mit CSS. Um beim Layout der Blöcke das gewünschte Ergebnis zu erzielen, habe ich mich für Flexbox entschieden. Danach beschäftigte ich mich mit der Anpassbarkeit der Seite für mobile Endgeräte und der Erstellung eines mobilen Menüs mittels JavaScript. Für einige Blöcke habe ich eine Animation angewandt, um diese Seite attraktiver zu gestalten. Anschließend müsste ich nur noch meine Seite auf GitHub hochladen, um auf deren Servern sie sich jetzt befindet.
Danke für ihre Aufmerksamkeit!
